# POD-JSON Specification

**POD-JSON** is an open JSON format for **print-on-demand products**, starting with apparel.

Goals:

- AI-friendly (easy to generate from prompts)
- Production-ready (RIP / DTG / DTF / screen)
- Platform-neutral (Shopify, custom shops, marketplaces)
- Human-readable & machine-validated (via JSON Schema)

## Repository structure

- `spec/v1.0/pod-json-v1.0.md` – human-readable spec
- `schemas/pod-order.v1.json` – JSON Schema for validators
- `examples/` – example POD-JSON files
